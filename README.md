# Cinatomy Movie API

A REST API built with FastAPI and SQLAlchemy, serving experiential movie profiles from the [Cinatomy dataset](https://www.kaggle.com/datasets/pomegrenade/cinatomy-experiential-movie-profiles) on Kaggle. The dataset contains subjective ratings across 25+ dimensions for movies from the IMDb Top 1000 list.

Built as a learning project to practice FastAPI, SQLAlchemy ORM, Pydantic schemas, and SQLite.

## Stack

- **FastAPI** — web framework
- **SQLAlchemy** — ORM and database interaction
- **Pydantic** — response validation and serialization
- **SQLite** — database
- **pandas** — data ingestion
- **uv** — package manager

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/first-api.git
cd first-api
```

### 2. Install dependencies

```bash
uv sync
```

### 3. Populate the database

Download the dataset CSV from Kaggle and place it in the project root, then run:

```bash
uv run python ingest.py
```

This only needs to be done once.

### 4. Start the server

```bash
uv run fastapi dev main.py
```

The API will be available at `http://127.0.0.1:8000`. Interactive documentation is at `http://127.0.0.1:8000/docs`.

## Endpoints

### General

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/` | Welcome message |
| GET | `/api/v1/movies` | List all movies (title and id only) |
| GET | `/api/v1/movies/id/{id}` | Get a single movie by id |
| GET | `/api/v1/movies/title/{title}` | Get a movie by title |
| GET | `/api/v1/movies/search` | Search with optional query parameters |

### Curated Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/movies/family_friendly` | Family friendly movies (rating >= 4) |
| GET | `/api/v1/movies/captivating` | Fast-paced and immersive movies |
| GET | `/api/v1/movies/underrated` | Original movies with high overall rating |
| GET | `/api/v1/movies/visual` | Visually stunning movies (visuals and effects == 5) |
| GET | `/api/v1/movies/sound` | Movies with excellent sound and no audio issues |
| GET | `/api/v1/movies/feelgood` | Feel-good movies with satisfying or earned endings |
| GET | `/api/v1/movies/background` | Easy-watch background movies |
| GET | `/api/v1/movies/trailermovie` | Movies whose trailers heavily spoil the plot |
| GET | `/api/v1/movies/challenging` | Cognitively demanding and highly original movies |
